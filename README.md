# Find_The_Number
🎯 Number Bubble Game
📌 Description

This is a simple and interactive browser game built using HTML, CSS, and JavaScript.
In this game, the user is given a target number, and 100 bubbles appear on the screen with different numbers. The player has to find and click the correct number to score points within a limited time.

🚀 Features
100 bubbles displayed on screen
Random number generation
Score increases by +10 for correct selection
Score decreases by -20 for wrong selection
⏱️ 60-second timer for gameplay
Simple and engaging UI
Fast-paced gameplay


🛠️ Tech Stack
HTML
CSS
JavaScript


🎮 How to Play
A target number will be displayed on the screen
You have 60 seconds to play
Look for the same number inside the bubbles
Click on the correct bubble
Score will increase by 10 points
If you click the wrong number, score will decrease by 20 points
Try to score as high as possible before time runs out ⏳
